MY LEARNING HIGHLIGHTS IN THE SUBJECT

My Computer Programming subject covers essential topics like programming solution design, fundamental elements of C, selection control structures, and iterative control structures. These lessons have equipped me with valuable skills. I can now design programming solutions using effective planning techniques, utilize fundamental programming concepts to create programs, incorporate conditional statements into my code, and develop programs using iterative statements to solve problems.

The most challenging part of this subject isn't a specific lesson but the major project we work on. Our instructor gives us a problem to solve using the lessons we've learned within a set timeframe. It's tough yet challenging because to solve the problem, we have to really think deeply about how to use what we've learned in class.
